# Linguagem_C

Aulas em linguagem C

Aulas gravadas para Youtube e acessados via ClassRoom do Google.
Ministradas pelo professor Reudismam.

Usamos na aula a linaguagem C, para materia de Algoritmo e Estruturas de Dados 1.

São 114 aulas em vídeo, com uma atividade no vídeo como exemplo, duas perguntas por vídeos e uma atividade para entregar via classroom.

A cada 4 vídeos ou terminando um determinado assunto, é necessário a produção de uma vídeo aula sobre o assunto ministrado do último vídeo dentre os 4.

Finalizando a produção de 16 vídeos-aulas sobre os assuntos ministrados na matéria.
